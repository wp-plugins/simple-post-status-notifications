Simple post status notifications sends emails to authors and editors when the post status changes.

== Description ==

Send post status notifications by email to Editors when posts are submitted for review and to Author when the Editor publishes or sends to draft.

= Rules =
Get a better editorial workflow by ensuring communication between your staff.

* Notification to editors with an email when the post status changes from Draft to Pending Review
* Notification to authors with an email when the post status changes from Pending Review to Published or Draft

= Features =

* Custom Email Subject and Message with Template-tags
* Template-tags help
* Multi language

= Translators =

* English (en_US) - Usagi Hoshino
* Spanish (es_ES) - Dientuki

== Installation ==

1. Upload the entire `simple-post-status-notifications` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
